PaperView PDF Export Example

Chapter 1: Introduction to PDF Export

PaperView now includes a powerful PDF export feature that allows you to generate PDF documents directly from your paginated content. This feature works in both browser and Node.js environments.

The browser implementation uses the native print API to generate PDFs, while the Node.js version leverages Puppeteer for server-side PDF generation with full control over output options.

Chapter 2: Features

  • Browser-based PDF export using window.print() API
  • Node.js PDF generation with Puppeteer
  • Custom page sizes (A4, Letter, Legal, etc.)
  • Landscape and portrait orientation
  • Headers and footers support
  • Progress callbacks for large documents
  • Metadata support (title, author, subject)
  • Print background colors and images

Chapter 3: Browser Usage

To export a PDF in the browser, simply call the exportPDF() method on your Previewer instance after rendering: 

const previewer = new PaperView.Previewer();
await previewer.preview(content, renderTo, stylesheets);

// Export with options
await previewer.exportPDF({
    filename: 'my-document.pdf',
    metadata: {
        title: 'My Document',
        author: 'John Doe'
    },
    onProgress: ({ stage, progress }) => {
        console.log(`${stage}: ${progress}%`);
    }
});

Chapter 4: Node.js Usage

For server-side PDF generation, use the NodePDFExporter class: 

const { NodePDFExporter } = require('paperview/node-pdf');

// From HTML string
const pdf = await NodePDFExporter.fromHTML(html, css, {
    format: 'A4',
    landscape: false,
    margin: {
        top: '2cm',
        bottom: '2cm'
    }
});

// Save to file
await NodePDFExporter.save(pdf, 'output.pdf');

Chapter 5: Advanced Options

The PDF exporter supports a wide range of options to customize the output: 

// Browser options
await previewer.exportPDF({
    filename: 'document.pdf',
    metadata: {
        title: 'Document Title',
        author: 'Author Name',
        subject: 'Document Subject'
    },
    onProgress: (progress) => {
        updateProgressBar(progress);
    }
});

// Node.js options
const pdf = await NodePDFExporter.fromHTML(html, css, {
    format: 'Letter',
    landscape: true,
    scale: 1,
    displayHeaderFooter: true,
    headerTemplate: '<div style="text-align: center;">Header</div>',
    footerTemplate: '<div>Page <span class="pageNumber"></span></div>',
    printBackground: true,
    margin: {
        top: '3cm',
        right: '2cm',
        bottom: '3cm',
        left: '2cm'
    },
    pageRanges: '1-5',
    preferCSSPageSize: true
});
